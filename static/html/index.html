What is Pomelo?

Discord is likely to remove the current Username#Tag system.
Instead of it we are likely going to get a similar system to Twitter with @Usernames and Display names.
This will not allow duplicate @Usernames but will allow duplicate Display names.
The API to check if a Username is valid or taken has already been exposed.
This website is a simple tool to check if a Username is valid or taken.
